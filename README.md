📢 آموزش رایگان: دانلود هر فایل فیلترشده با گیت‌هاب (بدون نیاز به فیلترشکن)

اگر به سایت‌هایی مثل archive.org یا هر سایت دیگری که فیلتر است دسترسی ندارید، با این ترفند ساده می‌توانید فایل‌های خود را دانلود کنید.

✅ پیش‌نیازها:
- یک حساب کاربری در گیت‌هاب (github.com) بسازید (رایگان است).
- یک مرورگر معمولی (کروم، فایرفاکس، مرورگر گوشی).

🔧 مراحل گام به گام:

1️⃣ پروژه آماده را فورک (کپی) کنید:
   - به لینک زیر بروید:
     https://github.com/HamidPalang/github-file-downloader
   - روی دکمه «Fork» در بالای راست صفحه کلیک کنید.
   - صبر کنید تا یک کپی از پروژه در اکانت خودتان ساخته شود.

2️⃣ وارد اکانت فورک شده خود شوید:
   - بعد از فورک، خودکار به صفحه پروژه‌ی کپی شده هدایت می‌شوید.
   - آدرس آن شبیه این است: https://github.com/YourUserName/github-file-downloader

3️⃣ اجرای عملیات دانلود:
   - روی تب «Actions» کلیک کنید.
   - در سمت چپ، روی «دانلود فایل دلخواه شما» کلیک کنید.
   - دکمه آبی «Run workflow» را بزنید.
   - یک کادر باز می‌شود. در آن، لینک مستقیم فایل مورد نظر خود را جایگذاری کنید.
   - روی دکمه سبز «Run workflow» کلیک کنید.

4️⃣ دریافت فایل دانلود شده:
   - بعد از چند ثانیه (حدود 30 ثانیه)، کنار کلمه run یک دایره سبز ظاهر میشود
   - بالای صفحه روی ریپازوتری اصلی برای مثال اینجا ( github-file-downloader) کلیک کنید
   - روی دکمه Code کلیک و گزینه دانلود زیپ رو بزنید
   - فایل دانلود شده رو از حالت زیپ خارج کنین و از فایلتون استفاده کنین

💡 نکات مهم:
- حتماً از «لینک مستقیم» فایل استفاده کنید (آدرس فایل باید به .zip، .apk، .mp4، .pdf و ... ختم شود).
- حجم فایل نباید بیشتر از ۵۰۰ مگابایت باشد (محدودیت گیت‌هاب).
- برای فایل‌های بزرگتر از ۵۰۰ مگابایت، این روش کار نمی‌کند.
